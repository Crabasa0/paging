Detailed Description

The interface for the MMU module.

Author
Prof. Scott F. Kaplan
Date
Fall 2018

A simple module that is part of the vmsim library. This module implements the simulated memory management unit (MMU) that typically provides hardware mapping of virtual-to-physical addresses.

void mmu_init ( vmsim_addr_t  upper_pt_addr)

Initialize the MMU.

Parameters
upper_pt_addrThe real base address of the upper page table.

This function stores the given real address of the upper page table. Doing so is analogous to setting a hardware MMU's page table register (PTR) with the physical base address of the upper PT.

Translate a simulated address into a physical address.

Parameters
sim_addrThe simulated address to be translated.
write_operationWhether the data at the given address is being read (false) or written (true).
Returns
the real address to which the simulated address maps.

This function walks the multi-level page table to find the mapping from the given simulated address to its corresponding real address. If the simulated address is not yet mapped to a real address, then this function calls vmsim_map_fault() (mimicking an address translation interrupt, or page fault, in hardware) to have the mapping created, and then restarts the translation.
